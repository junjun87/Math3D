# Math3D — AI 数理化拍照搜题平台

> 📱 拍照上传题目 → AI 识别 → 精确计算 → 交互式课件 → 离线可回看
>
> 支持**立体几何、解析几何、代数、化学**四大科目，覆盖 **26 种题型**。

---

## 一、核心功能

### 1.1 题目提交方式

| 方式 | 说明 |
|------|------|
| 📷 **拍照上传** | 手机拍照 → PaddleOCR 文字识别 → 自动结构化 |
| ⌨️ **文字输入** | 直接输入题目文字，跳过 OCR |

### 1.2 智能识别

使用 **DeepSeek LLM** 自动解析题目，识别科目和题型，提取已知条件，输出结构化 JSON 供计算内核使用。

支持 **4 大学科、26 种题型**：

#### 🔷 立体几何（7 种几何体 × 6 种题型）
- **几何体**：正方体、长方体、四棱锥、三棱柱、四面体、圆柱、圆锥
- **题型**：线面角、二面角、异面直线夹角、点面距离、体积、表面积
- **解题方法**：坐标法 + 向量法，SymPy 符号计算，杜绝浮点误差

#### 🔷 解析几何（7 种题型）
直线方程、圆的方程、椭圆方程、双曲线方程、抛物线方程、距离计算、交点求解

#### 🔷 代数（7 种题型）
一元一次方程、一元二次方程、方程组、不等式、数列（等差/等比）、因式分解、函数性质分析

#### 🔷 化学（6 种题型）
化学方程式配平、物质的量计算、浓度计算、pH 计算、气体定律、化学计量

### 1.3 精确计算

所有数值计算使用 **SymPy 符号引擎**，输出精确值而非浮点近似：

- `1/3` 而不是 `0.333...`
- `√2/2` 而不是 `0.707...`
- 分步展示：坐标系建立 → 公式代入 → 结果

### 1.4 交互式课件

| 科目 | 课件形式 |
|------|----------|
| 立体几何 | **3D 交互**（Three.js）：旋转/缩放/顶点标签/面高亮/分步动画 |
| 解析几何 | 分步解析 + KaTeX 公式渲染 |
| 代数 | 分步解答 + KaTeX 公式渲染 |
| 化学 | 分步解答 + KaTeX 公式渲染 |

所有课件为**独立 HTML 文件**，可下载、分享、离线打开。

### 1.5 PWA 支持（手机端）

- 📲 **添加到主屏幕**：像原生 App 一样使用
- 🔌 **离线访问**：看过的课件断网也能回看
- ⚡ **快速加载**：Service Worker 缓存静态资源

## 二、技术架构

```
┌──────────────────────────────────────────────────────┐
│                     前端 (React + Vite)                 │
│         拍照/输入 → 确认页面 → 课件展示                   │
└────────────────────────┬─────────────────────────────┘
                         │ HTTP API
┌────────────────────────▼─────────────────────────────┐
│                  后端 (FastAPI + Celery)                │
│   ┌──────────┐  ┌───────────┐  ┌──────────────────┐  │
│   │ LLM 解析  │→│ SymPy 计算 │→│ HTML 课件渲染      │  │
│   │ (4科识别) │  │ (精确求解) │  │ (3D/2D 交互)     │  │
│   └──────────┘  └───────────┘  └──────────────────┘  │
└────────────────────────┬─────────────────────────────┘
                         │
┌────────────────────────▼─────────────────────────────┐
│               基础设施 (Docker Compose)                  │
│   PostgreSQL 16  │  Redis 7  │  PaddleOCR  │  Nginx    │
└──────────────────────────────────────────────────────┘
```

### 数据流

```
拍照上传 → OCR识别 → 用户确认(可修正) → LLM结构化 → SymPy计算 → 课件渲染 → 3D/2D展示
```

## 三、部署方式

### 3.1 Docker 部署（推荐）

```bash
git clone https://github.com/junjun87/Math3D.git
cd Math3D
cp .env.example .env         # 编辑 .env 填入 LLM API Key
docker compose up -d          # 启动全部服务
```

### 3.2 服务端口

| 服务 | 端口 | 说明 |
|------|------|------|
| 后端 API | 8000 | FastAPI |
| 前端 | 5173 | React 开发服务器 |
| 数据库 | 5432 | PostgreSQL |
| 消息队列 | 6379 | Redis |
| OCR | 8001 | PaddleOCR |

### 3.3 配置说明

编辑 `.env` 文件：

```env
POSTGRES_DB=math3d
POSTGRES_USER=math3d
POSTGRES_PASSWORD=math3d_secret
LLM_API_KEY=your-deepseek-api-key       # DeepSeek API Key
LLM_API_BASE=https://api.deepseek.com/anthropic
LLM_MODEL=claude-sonnet-5
```

## 四、API 关键端点

| 方法 | 路径 | 说明 |
|------|------|------|
| `POST` | `/api/v1/problems/upload` | 上传图片题目 |
| `POST` | `/api/v1/problems/text` | 提交文字题目 |
| `GET` | `/api/v1/problems/{id}/ocr` | 轮询 OCR 结果 |
| `POST` | `/api/v1/problems/{id}/confirm` | 确认/修正 OCR → 触发计算 |
| `GET` | `/api/v1/problems/{id}/lesson` | 获取课件结果 |
| `GET` | `/api/v1/lessons/{id}/view` | 查看课件 HTML |
| `GET` | `/api/v1/lessons/{id}/download` | 下载课件文件 |
| `GET` | `/api/v1/health` | 健康检查 |

## 五、示例

### 立体几何

> **输入**：正方体 ABCD-A1B1C1D1，棱长为 2，求二面角 A1-BD-C1 的大小。
>
> **输出**：cosθ = 1/3，3D 正方体可旋转查看，顶点标签清晰显示，分步解析完整。

### 解析几何

> **输入**：求过点 (2,1) 且垂直于直线 3x-4y+5=0 的直线方程。
>
> **输出**：4x + 3y - 11 = 0，分步展示斜率计算、点斜式代入。

### 代数

> **输入**：解方程 x² - 5x + 6 = 0。
>
> **输出**：x = 2 或 x = 3，展示判别式、求根公式代入过程。

### 化学

> **输入**：配平化学方程式 Fe + O₂ → Fe₃O₄。
>
> **输出**：3Fe + 2O₂ → Fe₃O₄，展示元素守恒方程的矩阵求解过程。
